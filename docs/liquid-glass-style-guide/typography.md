# HIG: Typography (Tipografía / Apple 2025)

La tipografía en 2025 ha sido refinada para mantener la legibilidad contra los fondos dinámicos y translúcidos del sistema Liquid Glass.

## Principios de Tipografía
- **Bolder & Structured:** Las fuentes del sistema (SF Pro) son ahora **más gruesas** y se alinean frecuentemente a la **izquierda** para mejorar la escaneabilidad en áreas clave como alertas, encabezados y onboarding.
- **Dynamic Type:** El soporte para Dynamic Type es obligatorio para asegurar la accesibilidad en todos los niveles de visión.
- **Contraste Elevado:** El texto debe estar "anclado" con ratios de contraste altos (mínimo 4.5:1, idealmente 7:1 para texto pequeño) para evitar que se pierda con el movimiento del fondo.

## Aplicación en Borisao Wiki
- Usamos `font-bold` para todos los encabezados principales para asegurar que el peso tipográfico domine la refracción del cristal.
- Alineación a la izquierda en el **Banner del Archivo Central** para facilitar la lectura rápida.
- Uso de `text-white/90` o `text-white` para garantizar el contraste sobre el cristal abisal.
