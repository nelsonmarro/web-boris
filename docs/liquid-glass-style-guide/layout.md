# HIG: Layout (Diseño y Distribución)

Un buen diseño utiliza el espacio negativo, la alineación y el agrupamiento para priorizar lo esencial.

## Principios de Layout
- **Jerarquía Visual:** Usa tamaños de fuente contrastantes y pesos para guiar el ojo del usuario.
- **Áreas Seguras (Safe Areas):** Respeta las zonas seguras del sistema para evitar que el contenido se corte por muescas (notches) o indicadores de inicio.
- **Adaptabilidad:** Diseña para que la interfaz pase fluidamente entre modos vertical y horizontal (Multitarea).
- **Full-Bleed:** Extiende el arte y los fondos hasta los bordes exactos de la pantalla para una experiencia inmersiva.

## Aplicación en Borisao Wiki
- El **Banner del Archivo Central** usa un diseño "Full-Bleed" que llena el ancho de la sección de cristal líquido.
- Usamos `max-w-7xl` para asegurar que el contenido se vea centrado y bien distribuido en pantallas grandes.
