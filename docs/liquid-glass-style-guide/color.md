# HIG: Color (Color)

El color comunica la personalidad de tu aplicación y ayuda a indicar interactividad y jerarquía.

## Principios del Color
- **Interactividad:** Usa un "color de tinte" consistente (ej. Naranja Borisao) para todos los elementos accionables.
- **Contraste:** Mantén un ratio de contraste de al menos **4.5:1** (WCAG AA) para asegurar la legibilidad del texto.
- **Colores Dinámicos:** Utiliza colores semánticos que se adapten automáticamente a modos Claro y Oscuro.
- **Wide Color (P3):** Diseña para dispositivos con pantallas de gran gama de colores para obtener tonos más vibrantes y saturados.

## Especificaciones de Borisao Wiki
- **Color Primario:** Naranja Vibrante (`oklch(0.70 0.20 45)`).
- **Contraste:** Todo el texto sobre cristal debe ser blanco puro (`oklch(0.98 0.01 240)`) para maximizar la legibilidad en entornos oscuros.
- **Acentos:** Usa cian o azul cristalino para elementos secundarios (`oklch(0.75 0.20 240)`).
