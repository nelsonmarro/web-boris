# HIG: Toolbars (Barras de Herramientas)

Las barras de herramientas proporcionan acceso rápido a comandos frecuentes relevantes para la vista actual.

## Funcionalidad de Toolbars
- **Ubicación:** Superior (macOS/iPadOS) o Inferior (iOS).
- **Diseño Limpio:** Usa símbolos reconocibles y simples.
- **Evitar Mezcla:** No mezcles texto y símbolos en el mismo grupo de acciones.
- **Menús de Más:** Usa el icono de puntos suspensivos para ocultar acciones secundarias y evitar el hacinamiento visual.

## Aplicación en Borisao Wiki
- **Wiki Navigation:** Los botones de categorías actúan como una barra de herramientas secundaria para filtrar contenido de manera rápida y directa.
- **Acciones Rápidas:** Añadir iconos como `Share`, `Edit`, `Save` en la parte superior del artículo de la Wiki.
