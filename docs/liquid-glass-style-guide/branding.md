# HIG: Branding (Identidad de Marca / Apple 2025)

El branding debe sentirse integrado orgánicamente en el sistema en lugar de chocar con él.

## Principios de Branding
- **Harmonious Expression:** La marca debe "tocar en la misma clave musical" que el sistema.
- **Layered App Icons:** Los iconos soportan ahora tintes **Claroscuros y Transparentes (Clear)**. El modo Clear permite que el fondo use materiales Liquid Glass para integrarse en la pantalla de inicio.
- **Icon Composer:** Herramienta para crear iconos con capas que reaccionan a la luz y el movimiento, aportando "profundidad de marca".
- **Adaptive Capsules:** Se recomienda el uso de cápsulas adaptativas para los controles de marca que ajustan su tono según el entorno.

## Aplicación en Borisao Wiki
- El logo principal flota en una **cápsula adaptativa** en el Navbar.
- Usamos el gradiente de marca (Naranja Abisal a Blanco) en los encabezados principales del Archivo Central para mantener el reconocimiento sin sobrecargar la UI.
